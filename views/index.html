URL Shortener Microservice

freeCodeCamp API Project

About

This is a simple URL Shortener Microservice built with Express.

How to use

POST a URL to indigo-tiglon.glitch.me/api/shorturl/new and API will return a shortened URL in the JSON response.

{"original_url":"www.google.com","short_url":1}

If a invalid URL that doesn't follow http(s)://www.example.com(/more/routes) format send to the API, the JSON response will contain an error like {"error":"invalid URL"}

Visiting the shortened URL, will redirect you to the original link.
